# ExceptionHub

ExceptionHub will take your applications exceptions as they are thrown
and will log them in Github issues.

## Installation

Add this line to your application's Gemfile:

    gem 'exception_hub'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install exception_hub
